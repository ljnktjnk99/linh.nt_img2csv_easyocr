# OCR課題

table.pngにある表をcsvとして出力してください。

## 進め方

- ご自身のアカウントでGithubにリポジトリを作成してください。
- 作成したリポジトリに画像をcsvに変換するコードを格納してください。
- 変換スクリプトの使用方法などのリポジトリに関する説明はREADME.mdに記載してください。
- 完成したら、リポジトリのURLを共有してください。

## 要件

- OCRはeasyocrを使用してください。
  - 精度が悪いですが、その精度を上げる必要はありません。
- 変換スクリプトはPythonで記述してください。
- 必要なライブラリは適宜追加してください。
  - VLMなどを用いてcsvにすることは禁止です。
- 変換はコマンドラインで実行できるようにしてください。
